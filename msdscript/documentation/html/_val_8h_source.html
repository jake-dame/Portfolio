12/******************************************************************************
13 * \class Val
14 * \brief An abstract, base class representing a the value of an expression
15 *
16 * The Val class is an abstract class that defines regular and
17 * virtual functions that can be used to represent two mathematical
18 * values -- boolean values (BoolVal class) and integer values (NumVal class)
19 * -- of mathematical expressions (Expr class in Expr.h); calling interp() on
20 * an Expr object returns a Val object.
21 *
22 * The Val class handles base-level addition ( add_to() )and multiplication
23 * ( mult_with() ); it supports comparison between Val objects ( equals() ),
